Ce site permet de pouvoir avoir vos moyennes de chauqes élève sur base de leurs côtes.

Veiller introduire le nom de la classe ensuite le nom des élèves, le nombre de côtes à introduire

Dans les case des cotes vous pouvez choisir la note de lévaluation et introduire la cote de lélève

Le site affichera si l'élève a réussi grace à un code couleur rouge pour rater et vert pour réussi il vous seras aussi possible de mêtre un commantaire pour lélève

Pour finir il sera possible de sauvegarder les données sur un excel, vous pourrez choisir ou le fichier devrat être placer
